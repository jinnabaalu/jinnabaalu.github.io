Web-Based LDAP Directory Management

Create users, groups, and organizational units with an intuitive web interface. No more fighting with ldapadd and LDIF files.

LDAP Manager is an open-source product built by Jinna Balu under VibhuviOiO, born from years of managing LDAP directories in production environments — including multi-master deployments, migrations, and access control.

Features

Built for teams managing OpenLDAP directories in production

User Management

Create, update, and delete LDAP users through a clean web interface. Manage passwords and attributes visually.

Group Management

Create groups, manage memberships, and organize your directory structure without writing LDIF files.

OU Management

Create and manage organizational units. Build your directory tree structure through a visual interface.

Modern UI

Built with React, shadcn/ui, and Tailwind CSS. Responsive, accessible, and designed for daily operations.
